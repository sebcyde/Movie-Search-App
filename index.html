ReadMe

Thanks for taking the time to check out my movie search app! Here are a few bit of info about it.

Skills Used

  • HTML
  • CSS/SASS
  • Bootstrap (for this modal)
  • JS
  • Axios (for APIs)
  • Git / Github (Version control and repo hosting)
  • Responsive Design

Difficulties

  • JavaScript Modules

    • Solution: I kept forgetting the exact syntax of the import/export functions in JavaScript, so using this app (along with a few others) as practice, I have it pretty much ingrained now.

  • Integrating APIs

    • Solution: There were a number of different ways I could have done this (for example using Fetch, but i decided to use Axios as not only is the syntax cleaner and more compact, but it automatically parses the JSON data making the integration process easier and smoother.
